Frequently Asked Questions

Are mods supported?
Yes! ScalaCube features a 1-click installer for over 1,000 modpacks. It fully supports Forge, Fabric, and Steam Workshop integration for games like ARK and Rust, plus full FTP access for custom setups.
Can I change maps on my server?
Easily. You can switch between official maps or upload your own custom worlds directly through the control panel. We also provide full file access to manage your save data at any time.
Is crossplay supported?
Yes. We support crossplay for compatible titles. For Minecraft, you can enable Java and Bedrock players to join the same server using our pre-configured plugins.
Is there a refund policy?
ScalaCube generally operates on a non-refundable basis once a server is live. To help you decide, we offer a free-tier server so you can test our performance and features before buying.
